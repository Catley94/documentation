**Observação: a tecla do mod no Archcraft Openbox é a tecla Super (também conhecida como tecla Windows)**

|Atalho de teclado|Função|
|--|--|
|**mod+Return**| Abre um terminal (alacritty) |
|**mod+f**| Abre o gerenciador de arquivos (thunar) |
|**mod+w**| Abre um navegador da Web (firefox) |
|**mod+e**| Abre um editor de texto (geany) |
|**mod+(1-5)**| Alterar espaços de trabalho |
|**mod+Shift+(1-5)**| Mova o aplicativo para um espaço de trabalho específico |
|**alt/mod+Tab**| Alternar aplicativos (estilo Microsoft Windows) |
|**mod+(Up/Down/Left/Right)**| Ajustar a janela aos cantos |
|**mod+k**| Encaixa a janela no canto superior direito |
|**mod+h**| Encaixa a janela no canto superior esquerdo |
|**mod+j**| Encaixa a janela no canto inferior esquerdo |
|**mod+l**| Encaixa a janela no canto inferior direito |
|**mod+alt+up**| Move a janela para cima |
|**mod+alt+down**| Move janela para baixo |
|**mod+alt+left**| Move janela para a esquerda |
|**mod+alt+right**| Move janela para a direita |
|**control+alt+right**| Redimensionar janela para a direita |
|**control+alt+left**| Redimensionar janela para a esquerda |
|**control+alt+down**| Redimensionar janela para baixo |
|**control+alt+up**| Redimensionar janela para cima |
|**mod+Shift+Left/Right**| Enviar aplicativo para o espaço de trabalho seguinte/anterior |
|**up+up+down+down+left+right+left+right+B+A**| Obtenha superpoderes |
|**alt+F1** | Abre o launcher de aplicativos |
|**mod+n** | Abre o menu de rede |
|**mod+x** | Abre o powermenu |
|**mod+m** | Abre o menu de música |
|**mod+i** | Abre o menu de internet |
|**mod+s** | Abre o menu de captura de tela |
|**mod+r** | Executa aplicativos como root |
|**mod+w** | Abre o menu windows |
|**mod+p**| Abre o seletor de cores |
|**Print** | Tira uma  screenshot |
|**mod+Print** | Tira uma  screenshot (5s de delay) |
|**Shift+Print** | Tira uma  screenshot (10s de delay)|
|**Control+Print** | Tira uma  screenshot da janela ativa |
|**Control+Alt+Print** | Tira uma  screenshot de uma area |
|**mod+v**| Abre o menu de tarefas |
|**Control+Alt+t**| Abre o xfce4-terminal |
|**Control+Alt+v**| Abre o vim |
|**Control+Alt+n**| Abre o neovim |
|**Control+Alt+r**| Abre o ranger |
|**Control+Alt+h**| Abre o htop |
|**Control+Alt+b**| Abre o bashtop |
|**alt+Control+l** | Inicia a tela de bloqueio |
|**control+shift+Escape**| Sai do openbox |
|**control+Shift+BackSpace**| Reinicia o openbox |
|**control+shift+R**| Reconfigura o openbox |
|**mod+Escape**| Seleciona a janela e feche-a (xkill) |
|**mod+c/q**| Fechar janela focada |
|**alt+F5**| Minimizar janela para barra de tarefas |
|**alt+F6**| Alternar Maximizar |
|**alt+F7**| Alternar decorações |
|**mod+d**| Mostrar area de trabalho |
|**alt+r**| Redimensionar janela |
|**alt+m**| Mover janela |
|**mod+Space**| Mostrar menu raiz |
|**control+alt+space**| Mostrar menu de aplicativos |
|**alt+space**| Mostrar menu do cliente |
