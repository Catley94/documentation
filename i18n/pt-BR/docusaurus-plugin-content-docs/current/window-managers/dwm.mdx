**Observação: a tecla do mod no Archcraft Dwm é a tecla Super (também conhecida como tecla Windows)**

|Atalho de teclado|Função|
|------------ | ------------- |
|**mod+Return** | Abre um terminal (st) |
|**mod+Shift+Return** | Abre um terminal em modo janela flutuante |
|**mod+Control+Return**| Abre um terminal (alacritty)
|**mod+Shift+f** | bre o gerenciador de arquivos (thunar) |
|**mod+Shift+e** | Abre um editor de texto GUI (geany) |
|**mod+Shift+w** | Abre um navegador da Web (firefox) |
|**alt+F1** | Abre o launcher de aplicativos |
|**mod+n** | Abre o menu de rede  |
|**mod+x** | Abre o powermenu |
|**mod+m** | Abre o menu de música |
|**mod+i** | Abre o menu de internet |
|**mod+s** | Abre o menu de captura de tela |
|**mod+r** | Executa aplicativos como root |
|**mod+w** | Abre o menu windows |
|**Print** | Tira uma  screenshot |
|**mod+Print** | Tira uma  screenshot (5s de delay) |
|**Shift+Print** | Tira uma  screenshot (10s de delay)|
|**Control+Print** | Tira uma  screenshot da janela ativa |
|**mod+Control+Print** | Tira uma  screenshot de uma area |
|**alt+Control+l** | Inicia a tela de bloqueio |
|**mod+p**| Abre o seletor de cores |
|**mod+Control+q**| Sai do dwm |
|**mod+Shift+r**| Recarrega dwm |
|**mod+Shift+equal**| Aumenta tamanho da borda |
|**mod+Shift+minus**| Diminui tamanho da borda |
|**mod+Shift+BackSpace**| Redefini as bordas |
|**mod+Shift+BracketLeft**| Aumenta tamanho dos gaps |
|**mod+Shift+BracketRight**| Diminui tamanho dos gaps |
|**mod+Shift+backslah**| Redefini os gaps |
|**mod+Shift+g**| Alternar gaps |
|**mod+Shift+i**| Aumentar gaps internos |
|**mod+Control+Shift+i**| Diminuir gaps internos |
|**mod+Shift+o**| Aumentar gaps internos |
|**mod+Control+Shift+o**| Diminuir gaps internos |
|**mod+c**| Fechar um janela |
|**mod+escape**| Fechar, mas com xkill |
|**mod+j/k(Or Left right arrow)** | Foco circular (para frente e para trás |
|**mod+shift+j/k(or left right arrow)**| Alternar master e slave |
|**mod+i** | Aumente o número de janelas principais |
|**mod+u**| Diminuir o número de janelas principais |
|**mod+h(or Control+Left)**| Encolher para a esquerda |
|**mod+l(or Control+Right)**| Encolher para a direita |
|**mod+b**| Alternar a barra |
|**mod+Tab**| Alternar para master |
|**alt+Tab**| Alternar para a última tag |
|**mod+d**| Ocultar janela (enviá-la para os reinos das sombras) |
|**mod+Shift+d**| Restaurar Janela (recupere-a dos reinos das sombras) |
|**mod+0**| Reset the UI |
|**mod+Shift+0**| Redefinir o tag layout |
|**Alt+Control+Space**| Abra o menu de layout |
|**Mod+Control+comma/period**| Percorra os layouts |
|**mod+Shift+Space**| Alternar modo flutuante |
|**mod+f**| Alternar modo tela cheia |
|**mod+space**| Ativar layout com guias |
|**mod+t**| Ativar layout tiled |
|**mod+g**| Ativar layout grid |
|**mod+Shift+m**| Ativar layout visão única |
|**mod+Shift+s**| Ativar layout espiral |
|**mod+Shift+t**| Ativar layout de pilha |
|**mod+Shift+c**| Ativar layout master centralizado |
|**mod+Shift+x**| Ativar layout tatame |
