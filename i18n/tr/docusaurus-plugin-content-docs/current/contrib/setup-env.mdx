# Sistemi Yapılandırın

Geliştirme yapmak için sisteminizi yapılandıralım...

- Sisteminize `nodejs`, `npm` ve `yarn` kurulumlarını yapınız :
```bash
$ sudo pacman -S nodejs npm yarn
```

- `documentation` deposunu Github'dan klonlayın: :
```bash
$ git clone https://github.com/archcraft-os/documentation
```

- Deponun ana klasörune gerekli `node modules` kurun :
```bash
$ cd documentation
$ npm install
```

- Yerel geliştirme sunucusunu çalıştırın​ :
```bash
$ yarn run start
```

- Yeni belgeler ekleyin, sayfaları düzenleyin ve işiniz bittiğinde web sitesini oluşturun :
```bash
$ yarn run build
```
:::note NOT

Web sitesinin başarıyla oluşturulduğundan emin olunuz. Oluşturulan web sitesini Github'a commit'lemeden (ya da pull request oluşturmadan) önce yerel geliştirme sunucusunuzda test ediniz.

:::
