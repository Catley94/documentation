**Note: the mod key in Archcraft hyprland is the Super Key (Aka: Windows Key)**

|Keybinding|Function|
|--|--|
|**mod + Return** | Opens a terminal (alacritty) |
|**mod + T** | Opens a terminal in fullscreen mode |
|**mod + Shift + Return** | Opens a terminal in floating mode |
|**mod + D**| Opens app launcher |
|**mod + M** | Opens network menu |
|**mod + R** | Opens runner menu |
|**mod + X** | Opens a power menu |
|**mod + M** | Opens the music menu |
|**mod + A** | Opens screenshot menu |
|**mod + B** | Opens bluetooth menu |
|**mod + Shift + F** | Opens file manager (thunar) |
|**mod + Shift + E** | Opens a text editor GUI (geany) |
|**mod + Shift + W** | Opens a web browser (firefox) |
|**mod + P** | Color Picker |
|**Ctrl + Alt + L** | Lockscreen |
|**Print** | Takes a screenshot |
|**Alt + Print** | Takes a screenshot (5s delay) |
|**Shift + Print** | Takes a screenshot (10s delay) |
|**Ctrl + Print** | Takes a screenshot of active window |
|**mod + Print** | Takes a screenshot of an area |
|**mod + C** | Kill an App |
|**mod + Q** | Kill an App |
|**Ctrl + Alt + Delete** | Quit Hyprland |
|**mod + D** | Enable Fullscreen |
|**mod + Space** | Toggle between floating and tiled |
|**mod + S** | Enable Psuedo tiled mode |
|**mod + Shift + P** | Toggle Pin |
|**mod + Shift + S** | Swap windows |
|**mod * Shift + Left/Down/Up/Right** | Send the window to the edge of the screen (Tiled and Floating) |
|**Ctrl + Alt + Left/Right** | Switch workspace |
|**Ctrl + Alt + Shift + Left/Right** | Move window between workspaces |
|**mod + Ctrl + F** | Toggle all float mode |
|**mod + Ctrl + S** | Toggle all pseudo mode |
|**mod + Tab** | Switch focus between active windows |
|**mod + Left/Down/Up/Right** | Move focus |
|**mod + Shift + {1-8}** | Send focused window to another workspace |
|**mod + Ctrl + Right/Down** | Expand the window |
|**mod + Ctrl + Left/Up** | Shrink the window |
|**mod + Alt + Left/Down/Up/Right** | Move active floating window |

To install hyprland, just run `sudo pacman -Syyu archcraft-hyprland`
