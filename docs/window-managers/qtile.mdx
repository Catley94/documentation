**Note: the mod Key in Archcraft qtile is the Super Key (Aka: Windows key)
and the mod1 key is the Alt Key**

|Keybinding|Function|
|--|--|
|**mod + Return** | Opens a terminal (alacritty) |
|**mod + Alt + Return** | Opens a terminal in fullscreen mode |
|**mod + Shift + Return** | Opens a terminal in floating mode |
|**Alt + F1** | Opens app launcher |
|**Alt + F2** | Opens app command runner |
|**mod + N** | Opens network menu |
|**mod + B** | Opens Bluetooth menu |
|**mod + X** | Opens a power menu |
|**mod + M** | Opens the music menu |
|**mod + S** | Opens screenshot menu |
|**mod + R** | Runs apps as root |
|**mod + Shift + F** | Opens file manager (thunar) |
|**mod + Shift + E** | Opens a text editor GUI (geany) |
|**mod + Shift + W** | Opens a web browser (firefox) |
|**Ctrl + Alt + V** | Opens a TUI text editor (vim) |
|**Ctrl + Alt + R** | Opens a TUI file manager (ranger) |
|**Ctrl + Alt + H** | Opens a TUI system resource monitor (HTOP) |
|**mod + P**| Color Picker |
|**Ctrl + Alt + L** | Launches the qtile lockscreen |
|**Print** | Takes a screenshot |
|**Ctrl + Print** | Takes a screenshot (5s delay) |
|**Shift + Print** | Takes a screenshot (10s delay)|
|**Ctrl + Shift + Print** | Takes a screenshot of active window |
|**mod + Print** | Takes a screenshot of an area |
|**mod + C/Q** | Kill an App |
|**mod + Ctrl + Q** | Quit qtile (goodbye frien ;-;) |
|**mod + Ctrl + S** | Restart qtile |
|**mod + Ctrl + R** | Reload qtile configuration |
|**mod + F** | Enable FullScreen |
|**mod + Space** | Toggle betweeen floating and tiled |
|**mod + Shift + Space** | Toggle between different layouts |
|**mod + Left/Down/Up/Right** | Switch between windows |
|**mod + Shift + Left/Down/Up/Right** | Send the window to the edge of the screen |
|**mod + Alt + Left/Right** | Switch between workspaces |
|**mod + Shift + Space** | Toggle between layouts |
|**mod + B** | Switch to last opened workspace |
|**mod + Tab** | Change focus to other window |
|**mod + Shift+{1-8}** | Send focused window to another workspace |
|**mod + Ctrl + Left/Right/Up/Down** | Expand/Shrink the window |
|**mod + Shift + R** | Toggle window resize mode and use (Left/Down/Up/Right) to resize the window |
|**mod + Ctrl + Return** | Reset all window sizes |

To install Qtile, just run `sudo pacman -Syyu archcraft-qtile`
